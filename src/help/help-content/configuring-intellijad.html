Configuring IntelliJad
There are two levels of configuration in IntelliJad - IDE-level, and project-level. Both are accessible via the settings panel, in the relevant sections.
IDE-level configuration
The IDE-level configuration persists across multiple projects, and is the most useful place to set up IntelliJad. In here, you can for instance define automatic decompilation to memory and then never have to change the settings to have this type of decompilation occur in all your projects.
Project-level configuration
Project-level decompilation applies only to the current project.
